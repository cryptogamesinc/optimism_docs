# OPスタックビルダーのためのオープンソースコードリポジトリ

[エコシステムリポジトリ](https://github.com/ethereum-optimism/ecosystem)は、OPスタック上でアプリケーションをデプロイするビルダー向けに特別に設計されたライブラリ、ユーティリティ、およびフォーク可能なコード例の包括的なリソースです。
このエコシステムリポジトリは、OPスタック上で作業するすべてのビルダーの開発プロセスを容易にし、迅速化するために開発されており、プロトコル自体のすべての詳細を習得する必要なくコミュニティに価値を提供するのに役立ちます。

## はじめに

最初のリリースには、[Superchain Token List](/chain/tokenlist)にリストされているETHおよび任意のERC20トークンをブリッジする方法を示す例の[ブリッジアプリケーション](https://github.com/ethereum-optimism/ecosystem/tree/main/apps/bridge-app)が含まれています。
このアプリケーションは、独自のブリッジを構築しようとしている人々のための参考資料として機能し、L1およびL2の両方のプロトコルとどのように相互作用するかをより明確に理解するのに役立ちます。

さらに、L1およびL2の両方とやり取りする開発者にとって不可欠な一般的なユーティリティを含むパッケージも提供しています。たとえば、スマートコントラクトの[ネットワークペア](https://github.com/ethereum-optimism/ecosystem/blob/main/packages/op-app/src/configs/networkPairs.ts)や[デプロイメントアドレス](https://github.com/ethereum-optimism/ecosystem/blob/main/packages/op-app/src/configs/deploymentAddresses.ts)を追跡する機能が含まれています。
