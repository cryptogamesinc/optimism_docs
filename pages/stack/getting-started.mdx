# OP Stackの始め方

**OP Stackは、Optimismを支える標準化された、共有のオープンソース開発スタックであり、Optimism Collectiveによって維持されています。**

<Callout type="info">
  [Optimism Developer Blog](https://blog.oplabs.co/)に登録して、SuperchainとOP Stackに関する最新情報を入手してください。
</Callout>

OP Stackは、Optimism Collectiveが管理・維持するさまざまなソフトウェアコンポーネントで構成されており、それらが一体となってOptimismの基盤を形成しています。
OP Stackは、EthereumとOptimismのエコシステムのための公共財として構築されています。
ロールアップやチェーンのデプロイメントの基本を含め、OP Stackチェーンの操作方法を理解するには、[Chain Operator guide](/builders/chain-operators/self-hosted)を訪れてください。OP mainnetをサポートするために知っておくべきことの概要を得るには、[exchange](/builders/cex-wallet-developers/cex-support)および[wallet](/builders/cex-wallet-developers/wallet-support)のガイドをチェックしてください。

## OP StackがOptimismを支える

OP Stackは、現在のOP Mainnetの背後にあるソフトウェアの形で、将来的にはOptimism Superchainとそのガバナンスの形で、Optimismを支えるソフトウェアセットです。

Superchainコンセプトの登場により、提案されたSuperchainエコシステム内で相互運用できる新しいチェーンを安全に作成することが容易になりました。
その結果、OP Stackは主に、新しいL2ブロックチェーンを作成するための共有された高品質で完全にオープンソースのシステムの作成に焦点を当てています。
共有標準に基づいて調整することで、Optimism Collectiveは同じソフトウェアを再構築する必要を避けることができます。

現在のOP Stackは、L2ブロックチェーンの作成プロセスを大幅に簡素化していますが、これはOP Stackの本質を定義するものではありません。
OP Stackは、Optimismを支えるすべてのソフトウェアです。
Optimismが進化するにつれて、OP Stackも進化します。

**OP Stackは、Optimismエコシステムの特定の層を定義するソフトウェアコンポーネントまたは既存の層内のモジュールとしての役割を果たすコンポーネントと見なすことができます。**
現在のOP Stackの中心はL2ブロックチェーンを運営するためのインフラですが、OP Stackは理論的には、ブロックチェーンの基礎となる層の上にあるツール（ブロックエクスプローラー、メッセージ伝達機構、ガバナンスシステムなど）にまで拡張されます。

層は一般的に、スタックの下部（データアベイラビリティレイヤーなど）に向かってより厳密に定義されますが、スタックの上部（ガバナンスレイヤーなど）に向かってより緩やかに定義されます。

## 今日のOP Stack

Optimism Bedrockは、現在のOP Stackのバージョンです。
Bedrockリリースは、プロダクション品質のOptimistic Rollupブロックチェーンを立ち上げるためのツールを提供します。
現時点では、OP Stackの異なるレイヤーのAPIはこのRollup構成に密接に結びついています。

今日のOP Stackは、共通のセキュリティ、通信レイヤー、および共通の開発スタック（OP Stack自体）を共有するL2の提案ネットワークである[Optimism Superchain](/stack/explainer)をサポートするために構築されました。
OP StackのBedrockリリースは、Superchainがローンチされたときに互換性のあるL2を簡単に立ち上げることができます。
Superchain対応のL2を立ち上げたい場合は、OP StackのBedrockリリースに基づくチェーンを運営するためのガイドをチェックしてください。

OP Stackのコンポーネントを変更して新しいL2システムを構築することは可能です。
OP Stackでの実験に興味がある場合は、このサイトの[OP Stack Hacksセクション](/builders/chain-operators/hacks/overview)をチェックしてください。
ただし、Bedrockリリースの時点では、OP Stackはこれらの変更をサポートするようには設計されておらず、コードベースで**ハッキング**することになります。
そのため、**現時点では、OP Stack Hacksに対する開発者サポートは限定的（あるいは全くない）ことを期待してください。**
OP Stack Hacksは、Optimism Superchainと互換性のないチェーンを作成する可能性があります。
楽しんでください、ただし自己責任で行い、**Superchainに参加したい場合はBedrockリリースに固執してください。**

## 明日のOP Stack

OP Stackは進化するコンセプトです。
Optimismが成長するにつれて、OP Stackも成長します。
今日、OP StackのBedrockリリースは、新しいL2 Rollupのデプロイプロセスを簡素化しています。
スタックの作業が進むにつれて、異なるモジュールをプラグインして構成することが容易になるはずです。
[Superchain](explainer)が形作られ始めると、異なるチェーンがシームレスに相互運用できるようにするメッセージパッシングインフラストラクチャを含むように、OP Stackも進化する可能性があります。
最終的には、OP StackはOptimismが必要とするものになります。

## OP Stackをさらに深く掘り下げる

OP Stackの世界に飛び込む準備はできましたか？

*   現在のOP Stackのリリースについてもっと知りたい場合は、Bedrockリリースのページをチェックしてください。
*   OP Stackをより深く理解したい場合は、[Design Principles](protocol/design-principles)と[Components Overview](components)から始めてください。
*   Superchainに参加したい場合は、[Getting Started guide](explainer)を参考にして最初のSuperchain対応のL2を立ち上げるか、OP Stackのコードベースに直接飛び込んで詳細を学んでください。

OP StackはEthereumの次のフロンティアです。あなたはすでにここにいるのですから、何を待っているのですか？
